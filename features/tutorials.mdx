## Overview

In-app tutorials provide interactive, step-by-step guidance directly within the platform. Instead of reading external documentation, you learn by doing—with tooltips, highlights, and prompts that guide you through each feature.

## How Tutorials Work

Tutorials overlay on top of the normal interface:

1. **Highlight** — The relevant UI element is spotlighted
2. **Explain** — A tooltip describes what it does and why
3. **Act** — You perform the action yourself
4. **Progress** — Move to the next step automatically

## Starting a Tutorial

- **Automatic** — First-time users are prompted to start the onboarding tutorial
- **Manual** — Click the **?** help icon and select "Start Tutorial" for the current mode
- **From Settings** — Access all available tutorials from your account settings

## Tutorial Features

| Feature | Description |
|---------|-------------|
| **Skip & Resume** | Exit anytime and pick up where you left off |
| **Mode-Specific** | Each mode has its own contextual tutorial |
| **Progress Tracking** | See which tutorials you've completed |
| **Non-Blocking** | Dismiss tutorials to explore freely |

<Tip>
  Tutorials adapt to your current context. Starting a tutorial in Experiment mode walks you through scenario creation; starting in Design mode focuses on model building.
</Tip>
