## Welcome to Veydra!

This guide will help you get started with Veydra's system dynamics modeling platform. In just a few steps, you'll be creating and running your first model.

### Step 1: Access the Platform

Navigate to your Veydra dashboard and sign in to your account.

<Note>
  If you don't have an account yet, contact your administrator or sign up for access.
</Note>

### Step 2: Create Your First Project

<AccordionGroup>
  <Accordion icon="plus" title="Create New Project">
    Click "Create New Project" from your dashboard and give your project a descriptive name like "My First Model".
  </Accordion>
  <Accordion icon="gear" title="Choose Model Type">
    Select the type of model you want to create:
    - **Basic Model**: Simple stock-and-flow structure
    - **Advanced Model**: Complex multi-stock systems  
    - **Template**: Start from a pre-built model
  </Accordion>
</AccordionGroup>

### Step 3: Enter the Model Playground

<AccordionGroup>
  <Accordion icon="play" title="Launch Model Playground">
    Once your project is created, you'll enter the **Model Playground** - the heart of Veydra where you build and test your models.
  </Accordion>
  <Accordion icon="sliders" title="Adjust Parameters">
    Use the **Model Controls** panel to:
    1. Set initial parameter values
    2. Mark parameters as "Active" for real-time adjustment
    3. Configure simulation settings
    
    <Tip>Parameters marked as "Active" can be adjusted while running simulations.</Tip>
  </Accordion>
</AccordionGroup>

### Step 4: Run Your First Simulation

<Accordion icon="rocket" title="Execute and Analyze">
  1. Click the "Run" button to execute your model
  2. Watch results appear in real-time in the chart area
  3. Adjust parameters and run again to compare scenarios
  4. Toggle "Compare Mode" to see baseline vs current scenario
  
  <Tip>Use the refresh button in Model Controls to reset all parameters to defaults while keeping them active.</Tip>
</Accordion>
