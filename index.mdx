## What is Veydra?

Veydra is a comprehensive system dynamics modeling and simulation platform that enables users to create, run, and analyze complex models. Whether you're working on business forecasting, environmental modeling, or any complex system analysis, Veydra provides the tools you need.

### 🚀 **Interactive Model Building**

- Multi-stock chart displays
- Baseline vs scenario comparison

- Data export capabilities
